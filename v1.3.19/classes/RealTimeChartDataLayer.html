fetchAllCandles

  • fetchAllCandles(from): Promise<Candle[]>

  • Fetch candles for the activeId and candleSize.

    Limitation: A maximum of 1000 candles can be fetched in a single request. Therefore, the 'from' parameter must be chosen so that the time range between 'from' and 'to' does not exceed 1000 * candleSize seconds.

    Formula: (to - from) <= 1000 * candleSize

    If 'to' is not provided, it defaults to the latest loaded candle or the current time.

    Example: If candleSize = 60 (1 minute), the time range between 'from' and 'to' must be less than or equal to 60,000 seconds (~16.6 hours).

    Parameters

    • from: number

      UNIX timestamp in seconds from which to fetch candles.

    Returns Promise<Candle[]>

fetchCandles

  • fetchCandles(to, countBack): Promise<Candle[]>

  • Fetches candles for the activeId and candleSize within a specified time range.

    This method should be called iteratively with a maximum of 1000 candles per call (countBack <= 1000). After each call, the "to" value should be updated to candles[0].from - 1 to fetch older data in steps. This approach allows backward pagination of historical candles while avoiding overload or data gaps.

    Parameters

    • to: number

      Unix timestamp (in seconds) representing the end of the time range (inclusive).

    • countBack: number

      Number of candles to fetch backward from the "to" timestamp.

    Returns Promise<Candle[]>

    Promise resolving to an array of Candle objects.
